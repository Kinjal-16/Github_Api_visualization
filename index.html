2. Pull Request Size

The graphs given below compares the overall pull request sizes made for a repository of an organization.
This metric is significant because the team making the pull request must ensure that the pull request size is small enough so that the reviewerâ€™s ability to detect defective code is maximized. Therefore, the repositories with highest percentages of overall pull request size is show in the fig 3 whereas the repositories with lowest percentages of overall pull request size is show in the fig 4.
This is calculated by the taking the mean of the percentage change in the head repo size from its corresponding base repo size over the pull requests made to each repository.
